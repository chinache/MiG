# 绘图1流程图重绘成果

本目录包含按原 PDF 内容与横向排版结构重新生成的可编辑流程图成果。仓库仅保留文本格式源文件与预览，PPTX 二进制文件请通过生成脚本在本地生成。

## 输出文件

- `flowchart_redesign.pptx`：16:9 宽屏 PPTX，节点、连线、括号与文字均为 PPT 可单独编辑对象；为避免提交二进制文件，请运行 `python3 tools/generate_flowchart.py` 在本地生成。
- `flowchart_redesign.drawio`：draw.io / diagrams.net 源文件，便于继续编辑。
- `flowchart_redesign.svg`：预览图，用于快速查看整体版式与视觉风格。

## 版式与内容说明

- 主流程保持从左到右结构：拓客策略、分层分类、客户分流、存款策略、贷款策略及对应客户节点。
- 底部保留“考核支撑系统”横向支撑区域，并连接“自建绩效系统”“客户星级大屏”。
- 视觉风格采用白色背景、蓝色主流程节点、绿色支撑区域、浅灰普通系统卡片、圆角矩形与统一线条。
